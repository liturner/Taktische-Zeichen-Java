Taktische-Zeichen-Swing

This is a maven artefact (jar) containing not only the symbols, but also classes from the java.desktop module and some tools for very easily getting a list of ImageIcons. For more information on the modules and classes, check out the Javadoc under Project Reports on the left.

Maven and Gradle dependency information is here.

Check out the Usage page for some hints on how to integrate this module into your code.
